## 🚀 Fitur Utama
* **Penggunaan Modular**: Komponen dibuat terpisah sehingga mudah di-`include` ke dalam aplikasi atau tema web PHP.
* **Integrasi Sidebar & Config**: Terintegrasi dengan sistem konfigurasi tema JWC (`jwc_theme_config.php`).
* **Dokumentasi Lengkap**: Setiap folder dilengkapi panduan langkah demi langkah penggunaan dan penempatan file.
